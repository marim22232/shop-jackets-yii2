# Магазин верхней одежды на Yii2

Экзаменационный проект: интернет-магазин верхней одежды.

---

## О проекте

Проект создан на основе базового шаблона Yii2 Basic Template. Реализован полноценный функционал управления категориями и товарами интернет-магазина верхней одежды.

Проект включает в себя административную часть для управления контентом и пользовательскую часть для просмотра товаров.

---

## Функционал

### Административная часть
- **Управление категориями**: создание, просмотр, редактирование, удаление категорий
- **Управление товарами**: создание, просмотр, редактирование, удаление товаров
- Автоматическая генерация ЧПУ (slug) из названия

### Пользовательская часть
- Главная страница с отображением списка последних товаров
- Детальная страница каждого товара
- Просмотр всех товаров в табличном виде
- Просмотр всех категорий
- ЧПУ-ссылки для удобной навигации и SEO

### Технические особенности
- Миграции для управления структурой базы данных
- ЧПУ (человекопонятные URL) через urlManager
- Виджет для отображения списка товаров на главной странице
- CRUD код сгенерирован через Gii
- PostgreSQL в качестве базы данных

## Установка и запуск

### Требования
- PHP 8.3 или выше
- PostgreSQL 14 или выше
- Composer
- Git

### Пошаговая установка

**1. Клонировать репозиторий**
```bash
git clone https://github.com/marim22232/shop-jackets-yii2.git
cd shop-jackets-yii2
2. Установить зависимости через Composer

bash
composer install
3. Создать базу данных в PostgreSQL

sql
CREATE DATABASE shop_db;
4. Настроить подключение к базе данных

Отредактируйте файл config/db.php:

php
<?php
return [
    'class' => 'yii\db\Connection',
    'dsn' => 'pgsql:host=localhost;port=5432;dbname=shop_db',
    'username' => 'postgres',
    'password' => 'ваш_пароль',
    'charset' => 'utf8',
];
5. Выполнить миграции для создания таблиц

bash
php yii migrate
6. Запустить встроенный веб-сервер

bash
php yii serve
7. Открыть в браузере

text
http://localhost:8080

Возможные проблемы и решения
Ошибка подключения к БД
Проверьте, запущен ли PostgreSQL

Проверьте правильность логина и пароля в config/db.php

Убедитесь, что база данных shop_db создана

Ошибка 404 при открытии страниц
Убедитесь, что в папке web/ есть файл .htaccess

Проверьте настройки urlManager в config/web.php

Ошибка при миграциях
Проверьте подключение к БД командой php yii migrate/history

ЧПУ не работает на OpenServer/XAMPP
Включите mod_rewrite в настройках Apache

Проверьте настройки виртуального хоста

Демонстрация работы
Основные сценарии использования
1. Добавление новой категории

Перейти: Управление → Новая категория

Заполнить поля: Название, Slug, Описание

Нажать "Save"

2. Добавление нового товара

Перейти: Управление → Новый товар

Выбрать категорию

Заполнить: название, цену, размер, цвет, материал, количество

Нажать "Save"

3. Просмотр товаров на сайте

Открыть главную страницу http://localhost:8080

Нажать "Подробнее" на любом товаре

4. Редактирование цены

Перейти: Товары

Найти нужный товар

Нажать кнопку "Update"

Изменить цену → "Save"
