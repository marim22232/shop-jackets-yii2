# Техническое задание на разработку интернет-магазина верхней одежды

## 1. Цель проекта

Разработка веб-приложения для управления каталогом верхней одежды.

## 2. Функциональные требования

### 2.1 Административная часть

- CRUD операции для категорий товаров
- CRUD операции для товаров
- Автоматическая генерация slug из названия

### 2.2 Пользовательская часть

- Просмотр списка товаров на главной странице
- Просмотр детальной информации о товаре
- ЧПУ-ссылки для SEO-оптимизации

### 2.3 Технические требования

- Фреймворк: Yii2 Basic
- PHP 8.3+
- PostgreSQL
- Миграции для управления схемой БД

## 3. Структура базы данных

### Таблица `categories`

| Поле | Тип | Описание |
|------|-----|----------|
| id | PK | Первичный ключ |
| name | string | Название категории |
| slug | string | Уникальный идентификатор |
| description | text | Описание |
| created_at | timestamp | Дата создания |

### Таблица `products`

| Поле | Тип | Описание |
|------|-----|----------|
| id | PK | Первичный ключ |
| category_id | FK | Внешний ключ на categories |
| name | string | Название товара |
| slug | string | Уникальный идентификатор |
| price | decimal | Цена |
| description | text | Описание |
| image | string | Изображение |
| size | string | Размеры |
| color | string | Цвет |
| material | string | Материал |
| stock | integer | Количество на складе |
| created_at | timestamp | Дата создания |
| updated_at | timestamp | Дата обновления |

## 4. Состав проекта

- Модели Category, Product
- CRUD контроллеры
- Виджет для отображения товаров
- ЧПУ маршрутизация
