\# Техническое задание на разработку интернет-магазина верхней одежды



\## 1. Цель проекта

Разработка веб-приложения для управления каталогом верхней одежды.



\## 2. Функциональные требования



\### 2.1 Административная часть

\- CRUD операции для категорий товаров

\- CRUD операции для товаров

\- Автоматическая генерация slug из названия



\### 2.2 Пользовательская часть

\- Просмотр списка товаров на главной странице

\- Просмотр детальной информации о товаре

\- ЧПУ-ссылки для SEO-оптимизации



\### 2.3 Технические требования

\- Фреймворк: Yii2 Basic

\- PHP 8.3+

\- PostgreSQL

\- Миграции для управления схемой БД



\## 3. Структура базы данных



\### Таблица categories

\- id (PK)

\- name (название)

\- slug (уникальный идентификатор)

\- description (описание)

\- created\_at (дата создания)



\### Таблица products

\- id (PK)

\- category\_id (FK)

\- name (название)

\- slug (уникальный идентификатор)

\- price (цена)

\- description (описание)

\- image (изображение)

\- size (размеры)

\- color (цвет)

\- material (материал)

\- stock (количество)

\- created\_at, updated\_at



\## 4. Состав проекта

\- Модели Category, Product

\- CRUD контроллеры

\- Виджет для отображения товаров

\- ЧПУ маршрутизация
